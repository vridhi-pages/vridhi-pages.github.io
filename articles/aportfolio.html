Steps For Creating Portfolio with HTML and CSS
  • Create a folder with name (ex. Portfolio ) and inside this folder, create css folder.
  • Create a index.html or index.htm file.
  • Create a styles.css file in css folder.

Now, Add the following CSS codes to your styles.css file.

/* Reset some default styles */
body, h1, h2, p, ul, li {
    margin: 0;
    padding: 0;
}

/* Set a background color and text color */
body {
    background-color: #f0f0f0;
    color: #333;
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

/* Header Styles */
header {
    background-color: #007bff;
    color: #fff;
    text-align: center;
    padding: 15px;
}

/* About Section Styles */
#about {
    padding: 20px;
    text-align: center;
    background-color: #fff;
}

#about img {
    max-width: 100%;
    border-radius: 5%; /* Makes the image round in corner, Put 50 If You Want Rounded*/
}

/* Skills Section Styles */
#skills {
    background-color: #fff;
    padding: 20px;
}

#skills ul {
    list-style: none;
}

#skills li {
    margin-bottom: 10px;
}

/* Projects Section Styles */
#projects {
    padding: 20px;
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
}

.project {
    flex-basis: calc(33.33% - 20px);
    margin-bottom: 20px;
    padding: 10px;
    background-color: #fff;
    box-shadow: 0 0 5px rgba(0, 0, 0, 0.2);
}

.project img {
    max-width: 100%;
}

/* Resume Section Styles */
#resume {
    padding: 20px;
    text-align: center;
}

#resume a {
    display: inline-block;
    padding: 10px 20px;
    background-color: #007bff;
    color: #fff;
    text-decoration: none;
    border-radius: 5px;
}

/* Contact Section Styles */
#contact {
    padding: 20px;
    text-align: center;
}

/* Footer Section Styles */
footer {
    background-color: #333;
    color: #fff;
    text-align: center;
    padding: 10px;
}

/* Media Queries for Responsiveness */
@media (max-width: 768px) {
    #projects .project 
        flex-basis: calc(50% - 20px);
    }
}

@media (max-width: 480px) {
    #projects .project {
        flex-basis: 100%;
    }
}

Note :) Images put in Root file (ex. Portfolio).
